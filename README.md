编写目标
-----------

设计出一套具体的个人博客网站，实现博文的分享与管理，项目的介绍与共享。主要实现以下内容：

访客： 阅读博文；评论博文；对博主留言；关注博主；发私信点赞与打赏。

博主：
博文发表、编辑、删除与浏览；博文分类、归档；博文评论管理；留言管理；个人信息维护；对其他博主关注。

管理员：用户管理；博文管理并可以推荐博文；自动实现敏感信息过滤。
